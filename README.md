# Online_Course_Registration_Management_System

## Project Overview
This project is a simple Course Management System developed using MySQL. It demonstrates database design, table creation, relationships, constraints, and data management operations.

## Features
- Course information management
- Student information management
- Course enrollment tracking
- Primary Key and Foreign Key relationships
- Data integrity using constraints
- Auto Increment functionality
- Unique and Check constraints

## Database Structure

### Course_Info
Stores course details such as:
- Course ID
- Course Name
- Credits
- Course Fee
- Duration (Weeks)

### Stud
Stores student information:
- Student ID
- Student Name
- Email
- Registration Date

### Enrollment
Tracks student enrollments:
- Enrollment ID
- Student ID
- Course ID
- Enrollment Date
